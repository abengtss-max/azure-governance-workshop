# Module 6: Practical Scenarios and Q&A

## Scenario 1: Monthly spend by department

1. If tags exist, group by `department` in Cost Analysis.
2. If tags do not exist, group by subscription as temporary approximation.

## Scenario 2: New or deleted resources this week

1. Open Activity log.
2. Filter operation by Create or Delete.
3. Export CSV.

## Scenario 3: Security and compliance posture

1. Open Defender Secure Score.
2. Open Policy Compliance.
3. Export and share monthly snapshot.

## Scenario 4: Who changed or deleted a resource

1. Open Activity log.
2. Filter by operation and time.
3. Inspect event details for caller identity.

## Scenario 5: Budget nearing limit

1. Open Cost Analysis.
2. Group by Resource.
3. Identify top cost drivers.

## Scenario 6: Who has elevated access

1. Open IAM Role assignments.
2. Filter Owner and Contributor.
3. Export report for audit.
