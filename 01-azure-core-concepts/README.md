# Module 1: Azure Core Concepts

Goal: Understand Azure hierarchy and the reporting impact of structure, naming, and tags.

## Step-by-step

1. Open https://portal.azure.com.
2. Go to Subscriptions and review available subscriptions.
3. Open one subscription and inspect its resource groups.
4. Open Management groups and review hierarchy.
5. Open a resource and check its Tags tab.

## Why this matters

- Subscription is a billing boundary.
- Resource groups improve ownership and reporting.
- Tags enable cost split by department, project, and environment.

## Recommended tags

- `environment`: prod, test, dev
- `department`: IT, Finance, HR
- `project`: app name
- `cost-center`: internal code
