# Module 5: Dashboards and Self-Service Reporting

Goal: Build simple, repeatable dashboards for leadership and operations.

## Step-by-step: Portal dashboard

1. Open Dashboard.
2. Create a new dashboard.
3. Pin Cost Analysis chart.
4. Pin Policy compliance chart.
5. Pin Defender Secure Score tile.
6. Pin Resource Graph count result.
7. Save and share the dashboard.

## Step-by-step: Workbooks

1. Open Monitor > Workbooks.
2. Use built-in templates (Activity, Cost, Alerts).
3. Save a custom workbook for recurring review.

## Step-by-step: Power BI

1. Connect to Azure Cost Management connector.
2. Import usage and budget data.
3. Build visuals: total spend, trend, top subscriptions, top resources.
4. Publish and schedule refresh.
