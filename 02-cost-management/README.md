# Module 2: Cost Management and Analysis

Goal: Analyze spend, set budgets, and export data for reporting.

## Step-by-step: Cost Analysis

1. Open Cost Management in the Azure portal.
2. Select scope (management group or subscription).
3. Set time range to Last 3 months.
4. Group by Subscription, then Service name, then Resource group.
5. Add filters as needed (location, resource type, tag).

## Step-by-step: Budgets

1. Open Budgets.
2. Create a monthly budget.
3. Set alerts at 80% and 100%.
4. Add email recipients.

## Step-by-step: Export

1. Open Exports.
2. Create scheduled monthly export to a Storage Account.
3. Use CSV export for Excel and Power BI.

## Key notes

- Use amortized cost for trend analysis.
- Use actual cost for billing reconciliation.
- Tag coverage is required for department-level reporting.
