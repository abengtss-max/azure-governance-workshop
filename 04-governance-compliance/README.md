# Module 4: Governance and Compliance

Goal: Use Azure Policy, Defender for Cloud, and IAM reports for governance visibility.

## Step-by-step: Policy Compliance

1. Open Policy.
2. Open Compliance view.
3. Set scope to management group.
4. Review compliant vs non-compliant resources.
5. Export compliance data.

## Step-by-step: Defender for Cloud

1. Open Defender for Cloud.
2. Review Secure Score.
3. Open Recommendations.
4. Filter High severity recommendations.

## Step-by-step: Access reporting

1. Open Access control (IAM) at subscription scope.
2. Review role assignments.
3. Filter roles: Owner and Contributor.
4. Download role assignments CSV.
